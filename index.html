Learning Path
おすすめの学習順序

初めて学ぶ方は、この順番で読むと理解が深まります。

1
🧠 行動経済学で「人が動く理由」を理解する

すべての説得・コピーライティング技術の土台。なぜ人は非合理な選択をするのかを知ることが、説得設計の出発点。システム1/2・プロスペクト理論・ナッジから始める。

行動経済学を学ぶ →
2
💡 SUCCESSで「伝わるメッセージ」の原則を掴む

アイデアを記憶に残す6原則。何かを伝えたいとき、どのように表現すれば相手の頭に残るかの基本を学ぶ。

SUCCESSを学ぶ →
3
✍️ PESONAでコピーライティングの「型」を身につける

6ステップのコピー構成フレームワーク。問題→共感→解決策→オファー→絞り込み→行動の流れを体で覚える。

PESONAを学ぶ →
4
🚀 PASBECONAで精度を上げる

PESONAを使いこなせたら、Benefit・Evidence・Contentsを加えた9ステップへ。高関与商品や長文セールスレターに対応できる応用型。

PASBECONAを学ぶ →
5
📋 PMMシートで書く前・書いた後を仕組み化する

コピーを書く前のリサーチと書いた後のチェックを仕組み化。23項目のリサーチシートと18項目のセルフチェックで、ミスを防ぎ品質を担保する。

PMMシートを学ぶ →
